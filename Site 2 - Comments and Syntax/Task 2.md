# Program 2 - PHP Comments and Syntax

### Technical Explanation

This task develops core PHP syntax knowledge: commenting styles, multi-line output, whitespace insensitivity, case sensitivity, and statement rules using semicolons.

---

## Task instructions

1. Open [comments.php](./comments.php) and identify examples of:
   - single-line comments with `#`
   - single-line comments with `//`
   - multi-line comments with `/* ... */`
2. Open [multiline.php](./multiline.php) and compare:
   - here-document output
   - a normal string printed over multiple lines
3. Open [whitespace.php](./whitespace.php) and explain why all three `$four` variables have the same value.
4. Open [casesensitive.php](./casesensitive.php) and explain why `$capital` and `$CaPiTaL` are treated differently.
5. Open [statements.php](./statements.php) and identify the statements ending in semicolons.
6. In [yourtask.php](./yourtask.php), add your own code to demonstrate all of the above concepts.

---

## Run in Codespaces

From the project root, run:

```bash
/usr/bin/php -S 0.0.0.0:8000 -t "Site 2 - Comments and Syntax"
```

Then open the forwarded port URL and browse to:

1. `/index.php`
2. `/yourtask.php`

### Stop the server

When you are finished, stop the server by pressing **Ctrl+C** in the terminal where it is running.
