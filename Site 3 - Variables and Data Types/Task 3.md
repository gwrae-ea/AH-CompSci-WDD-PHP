# Program 3 - Variables and Data Types

### Technical Explanation

This task explores PHP variables and the eight data types: integers, floats (doubles), booleans, NULL, strings, arrays, objects, and resources. Focus on the five simple types and demonstrate variable assignment, type conversion, and basic operations.

---

## Task instructions

1. Open [variables.php](./variables.php) and understand:
   - How variables are declared with `$`
   - How the = operator assigns values
   - Variables can be reassigned

2. Open [naming.php](./naming.php) and review:
   - Valid variable names (must start with letter or underscore)
   - Characters not allowed in variable names (+ - % ( ) . & etc)
   - Best practices for naming (camelCase, snake_case, UPPERCASE)
   - Automatic type conversion between numbers and strings
   - String concatenation
   - Type juggling in comparisons

3. Explore each simple data type:
   - [integers.php](./integers.php) — whole numbers
   - [floats.php](./floats.php) — decimal numbers
   - [booleans.php](./booleans.php) — true or false
   - [null.php](./null.php) — NULL type
   - [strings.php](./strings.php) — sequences of characters

4. Explore compound types:
   - [arrays.php](./arrays.php) — collections of values (indexed and associative)

6. In [yourtask.php](./yourtask.php), create variables for each of the five simple data types, assign values, and use them in output statements.

---

## Run in Codespaces

From the project root, run:

```bash
/usr/bin/php -S 0.0.0.0:8000 -t "Site 3 - Variables and Data Types"
```

Then open the forwarded port URL and browse to:

1. `/index.php`
2. `/yourtask.php`

### Stop the server

When you are finished, stop the server by pressing **Ctrl+C** in the terminal where it is running.
